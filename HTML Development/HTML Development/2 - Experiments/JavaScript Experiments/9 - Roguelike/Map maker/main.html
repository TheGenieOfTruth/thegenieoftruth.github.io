Map editor

The map editor is used to add tiles to the map. Use the +s on the side to increase the size of the map, and type inside of cells to set the flavor of cell. You can later edit the properties of the cells through the Cell editor. You can also right click to cycle through the cell flavors.

Cell editor

Here, you can change a cell's properties. For example, you can add text to a sign, toggle if a wall is a gate or not, and if so, is it on or off. Access a cell's data by clicking on it.
