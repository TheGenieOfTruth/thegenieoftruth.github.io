Web Design

In one minute

Web design takes several components:

  • HTML
  • CSS
  • JavaScript
  • A moderately smart person

How do you use them together?

HTML is basically a skeleton of a webpage. Its fairly useless when used by itself.

When CSS is added the the mix, things start to look nicer, and also become more useful.

JavaScript is used for a number of things. It can add CSS and HTML to a page, and can manipulate them.

And finally, you need a moderately smart person to actually make all three of these.
